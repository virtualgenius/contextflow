Key Changes

Visual Grouping

Subtle divider lines create 7 logical groups instead of ~20 flat items. Each group has a section header.

Consistent Property Rows

Code Size, Boundary, Role, and Team all use the same label-left/control-right layout with full-width selects (no more w-32).

Badges Get Labels

Classification and Evolution badges now have tiny "Domain" and "Evolution" labels above them, and sit on a single row.

Compact Toggle Grid

Legacy and Big Ball of Mud toggles share a 2-column row, saving vertical space.

Groups Get a Row

Group chips now have a "Groups" label in the property-row layout instead of floating without context.

Reordered for Workflow

Identity first, then strategic/org context, then technical detail, then notes/issues, then relationships. Most-edited fields at the top.
