What changed

  • Full-width, uniform cards. Every card is w-full with a two-row layout (name + actions, then badges + meta), so teams and repos read as the same component family at the same height.
  • Always-reachable panel. The sidebar (with its Add control) renders even at zero teams/repos, so you can set up your teams and repos up front instead of discovering them through a context.
  • Pinned chrome, scrolling list. Filter row pins to the top, Add control pins to the bottom; only the card list scrolls.
  • Repos are selectable. A repo card opens a Repo inspector (name, remote URL, owning teams, assigned context, delete), matching teams and contexts.
  • Filter by type / status. Teams filter by topology; repos filter by assigned/unassigned, alongside the existing name search.
  • In-place Add repo. Repos can be created from the sidebar, not only from a context inspector.

Colors use the real topology palette from src/lib/teamColors.ts. Dark mode omitted from the mockup; the implementation keeps the existing dark variants. Glyphs are placeholders for lucide icons (Crosshair, Trash2, ExternalLink, Search).
