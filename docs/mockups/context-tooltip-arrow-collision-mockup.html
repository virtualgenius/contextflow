Concept tooltip occludes the top connection arrow

GH #37 · bead contextflow-46o. On context hover, the rich concept tooltip is anchored just above the top edge of the box, exactly where the upstream arrow lives. The tooltip is a document.body portal at z 60; the arrow is sealed inside the React Flow canvas stacking context (root z 0), so the tooltip always paints over it. This is a layout collision, not a z-index one.
