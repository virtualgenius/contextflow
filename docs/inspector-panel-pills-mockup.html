Tradeoffs

Wins
  • Every option visible without a click
  • Boundary pills can preview the border style (dotted/thin/thick) the node will get
  • Code Size pills can hint at relative node size
  • One interaction pattern across all enum fields
Costs
  • Code Size's 5 pills crowd a 320px panel and wrap on narrower widths
  • Role labels are long; abbreviating ("Revenue" for "Revenue Generator") loses precision
  • Pill rows take more vertical space than a single-line dropdown
  • If we add a 5th boundary level or 6th size bucket, pills don't scale
Team picker stays a dropdown

Team is not a fixed enum (user-defined list, could be 20+), so it stays a select in both variants.
